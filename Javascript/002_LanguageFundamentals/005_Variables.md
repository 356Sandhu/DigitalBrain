# Variables

- named storage for data
- declared with the 'let' keyword

## Declaration

### Single Variable

```javascript
let name = "Satvir";
console.log(name);
// Prints: Satvir
```

### Multiple Variables

```javascript
let name = "Satvir",
  age = 22,
  country = "Canada";
// Prints: Satvir
```

## Assignment

You can assign variables to store more than just numbers and strings

```javascript
let name = "Satvir";
let author = name;
console.log(author);
// Prints: Satvir
```
